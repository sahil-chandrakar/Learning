‚úÖ 2. OR Operator: ||
Rule

A || B

If A is truthy, return A

If A is falsy, return B

This is HUGE in React for giving default values.

üîç Examples
Example 1:
console.log("" || "Default");
// "Default"


"" is falsy, so it returns "Default".

Example 2:
console.log("Hello" || "Default");
// "Hello"


"Hello" is truthy, so it returns "Hello".

Example 3:
console.log(0 || 100);
// 100


0 is falsy, so it returns 100.

Example 4 (React example):
const username = "";

return <h1>{username || "Guest"}</h1>;


Output ‚Üí Guest
Because "" is falsy.
