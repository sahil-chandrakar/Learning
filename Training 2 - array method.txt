filter()
-----------
filter() creates a new array containing only the items that pass a condition (return true).
It removes items that donâ€™t match the condition.
Original array stays the same.

ex:

find()
-----------
Returns the first item that matches a condition.

Key points

forEach()
-----------
Runs a function once for each item in the array.

Key points
Does not return anything 
Used for side effects (logging, updating variables)
Does NOT change original array unless you modify items manually,
if modified then original array will change

-----------
