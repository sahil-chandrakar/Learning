IMPORTANT POINTS
---------------

Events are actions happening in the browser (click, input, submit, keypress, scroll, change, etc.).

JS handles events using addEventListener(eventType, callback).

One element can have multiple listeners for the same event.

Inline events like <button onclick="fn()"> are not recommended.

Common event categories:

Mouse: click, dblclick, contextmenu, mouseover, mouseout

Keyboard: keydown, keyup, keypress

Form: input, change, submit, focus, blur

Window: load, scroll, resize

Event Bubbling â€“ Simple Definition
-------------

When an event triggers on a child, it automatically moves upward to its parent elements until the root.

Stop event from moving up:
event.stopPropagation();


Stop bubbling + stop other listeners on the same element:
event.stopImmediatePropagation();
