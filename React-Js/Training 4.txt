ğŸ”· 4. useNavigate (Navigate using JS)
-------------------
Used when you want to navigate using code, not a <Link>.

JS Example:
import { useNavigate } from "react-router-dom";

function Home() {
  const navigate = useNavigate();

  return (
    <>
      <h2>Home Page</h2>
      <button onClick={() => navigate("/about")}>
        Go to About
      </button>
    </>
  );
}

TS Example:
import { useNavigate } from "react-router-dom";

function Home(): JSX.Element {
  const navigate = useNavigate();

  return (
    <>
      <h2>Home Page</h2>
      <button onClick={() => navigate("/about")}>
        Go to About
      </button>
    </>
  );
}

â­ Explanation
useNavigate gives a function
calling navigate("/about") switches the page
no reload happens

ğŸ”· 5. Dynamic Routes + useParams
-----------------------
Used for pages like:

/users/10
/products/5
/blog/react-hooks

â­ JS Example:
Step 1: Declare dynamic route
<Route path="/user/:id" element={<User />} />

Step 2: Access parameter
import { useParams } from "react-router-dom";

function User() {
  const { id } = useParams();
  return <h3>User ID: {id}</h3>;
}

â­ TS Example:
import { useParams } from "react-router-dom";

function User(): JSX.Element {
  const { id } = useParams<{ id: string }>();
  return <h3>User ID: {id}</h3>;
}

â­ Why useParams?
----------------
Because URLs contain dynamic values â€” React Router extracts them for you.
