IMR: Passing function reference vs calling function
---------------------------

1ï¸âƒ£ Passing Function Reference
âœ” Definition
You give React the function itself, so React will call it later when the event happens.

âœ” Example

âœ” Whatâ€™s happening
-> handleClick is not executed now
-> React stores it
-> React executes it when user clicks
-> This is the correct way for most events

2ï¸âƒ£ Calling the Function
âœ” Definition
You execute the function immediately while rendering â€” React does NOT wait for the click.

âŒ Example

âŒ Whatâ€™s happening
handleClick() runs immediately during render
React gets the result (not a function)
The click handler breaks

3ï¸âƒ£ Using Arrow Function
âœ” Definition
Used when you want to pass arguments to your function.

âœ” Example
<button onClick={() => handleClick(10)}>Click</button>

âœ” Whatâ€™s happening
-> You give React a new function: () => handleClick(10)
-> That new function runs only on click
-> Inside it, your handleClick(10) is executed




ğŸ”· 6. State (useState)
------------------------
â“ What is state?
State = data that changes inside a component.

âœ” What it does
When state changes â†’ React automatically re-renders the component.

JS Example:
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <>
      <p>{count}</p>
      <button onClick={() => setCount(count + 1)}>+</button>
    </>
  );
}

TS Example:
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState<number>(0);

  return (
    <>
      <p>{count}</p>
      <button onClick={() => setCount(prev => prev + 1)}>+</button>
    </>
  );
}

ğŸ”· 8. Conditional Rendering
------------------------
â“ What is it?
Show UI based on a condition.

âœ” What it does
React chooses what to display.

JS Example:
{isLogged ? <p>Welcome</p> : <p>Please login</p>}


TS Example:
{isLogged ? <p>Welcome</p> : <p>Please login</p>}

(Same syntax, but isLogged would be typed as boolean.)

ex:
import { useState } from "react";

const App = () => {
  const [isLogged, setIslogged] = useState<boolean>(true);

  const handleClick = (e: React.MouseEvent<HTMLButtonElement>) => {
    e.preventDefault();
    setIslogged((prev) => !prev);
  };
  return (
    <>
      <button onClick={handleClick}>Toggle</button>
      {isLogged ? <h1>Welcome</h1> : <h1>Please Login</h1>}
    </>
  );
};
export default App;



ğŸ”· 9. List Rendering + keys
--------------------------------
â“ What is list rendering?

Rendering arrays inside JSX.

âœ” What keys do

Keys help React identify each item uniquely.

JS Example:

  return (
    <ul>

TS Example:

function App(): JSX.Element {
  return (
    <ul>
      {users.map((u, i) => (
        <li key={i}>{u}</li>
      ))}
    </ul>
  );
}
