What is TypeScript? (Very Simple Explanation)
---------------
TypeScript = JavaScript + Types.
JavaScript is flexible but sometimes too flexible:

let age = 20;
age = "twenty"; // JS allows this (bad)

TypeScript prevents mistakes like this.

let age: number = 20;
age = "twenty"; // âŒ Error in TypeScript

So TypeScript catches errors before running the code â†’ saves time.

5. Type Alias (very useful)
---------------------
You create your own type.

type User = {
  id: number;
  name: string;
};

let u1: User = { id: 1, name: "sahil" };
let u2: User = { id: 2, name: "pawan" };


6. Interface (same as type but for objects)
-------------------------
Used more in React!

interface Student {
  id: number;
  name: string;
}

let s1: Student = {
  id: 1,
  name: "mani",
};


7. Union Types (|)
---------------
